# 记忆化搜索

参考：[记忆化搜索](https://oi-wiki.org/dp/memo/)

记忆化搜索是一种通过记录已经遍历过的状态的信息，从而避免对同一状态重复遍历的搜索实现方式。

因为记忆化搜索确保了每个状态只访问一次，它也是一种常见的动态规划实现方式。
