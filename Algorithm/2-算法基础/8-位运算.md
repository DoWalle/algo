# 位运算

## 1、判断奇偶性

偶数的二进制的末尾 为0，奇数的二进制的末尾 为1

按位与 & 将参与运算的两操作数各对应的二进制位进行与操作，位数不够的数自动在前面补 0

- 偶数：原数的二进制末尾为0，与1进行按位与，结果为0
- 奇数：原数的二进制末尾为1，与1进行按位与，结果为1

```c++
#include <iostream>

using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        if (i & 1) {
            cout << i << ", " << (i & 1) << endl;  // 奇数
        } else {
            cout << i << ", " << (i & 1) << endl;  // 偶数
        }
    }
    // 0, 0
    // 1, 1
    // 2, 0
    // 3, 1
    // 4, 0
    return 0;
}
```

## 2、除二

注意：结果是向下取整

```c++
#include <iostream>
using namespace std;

int main() {
    cout << (1 >> 1) << endl;  // 0
    cout << (2 >> 1) << endl;  // 1
    cout << (3 >> 1) << endl;  // 1
    cout << (4 >> 1) << endl;  // 2
    cout << (5 >> 1) << endl;  // 2
    return 0;
}
```
