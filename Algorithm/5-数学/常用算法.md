# 常用算法

## 1、短除法

答案逆序数处

```c++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int num = 123;
    vector<int> res;
    while (num != 0) {
        res.push_back(num % 10);
        num /= 10;
    }
    for (int a : res) {
        cout << a << " ";
    }

    return 0;
}

```
