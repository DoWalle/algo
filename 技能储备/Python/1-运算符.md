## 位运算 

```python
&		# 与 
|   # 或
^   # 异或
~   # 取反
<<  # 左移
>>	# 右移
```

**异或的性质：**

- 异或不分前后
- 异或的相反运算还是异或
- 零与任意值异或，那个值不变，相当于乘法里的1
- 两个相同的值异或，值为0，因此一个数的偶数次异或为0

```python
a^b = c # 1^12=13
b^a = c # 12^1=13
c^b = a # 13^12=1 
c^a = b # 13^1=12

0^a = a # 0^14=14
a^0 = a # 14^0=14

a^a = 0 # 5^5=0
```
