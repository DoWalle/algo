#### [219. 存在重复元素 II](https://leetcode-cn.com/problems/contains-duplicate-ii/)

滑动窗口

```python
class Solution:
    def containsNearbyDuplicate(self, nums: List[int], k: int) -> bool:
        sz = len(nums)
        left = 0
        right = 0
        window = collections.defaultdict(int)

        while right < sz:
            window[nums[right]] += 1
            if window[nums[right]] == 2:
                return True
            while right - left == k:
                window[nums[left]] -= 1
                left += 1
            right += 1
        return False
```
