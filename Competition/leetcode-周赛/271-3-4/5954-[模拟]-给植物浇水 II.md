#### [5954. 给植物浇水 II](https://leetcode-cn.com/problems/watering-plants-ii/)

时间复杂度：`O(n)`

空间复杂度：`O(1)` 

```python
class Solution:
    def minimumRefill(self, plants: List[int], capacityA: int, capacityB: int) -> int:
        sz = len(plants)
        ans = 0
        remain_a = capacityA
        remain_b = capacityB
        zone = sz//2 if sz % 2 == 0 else sz//2+1

        for i in range(zone):
            if i == sz-1-i: # 当两个人相遇的时候
                if remain_a >= remain_b:
                    if plants[i] > remain_a:
                        remain_a = capacityA
                        ans += 1
                else:
                    if plants[sz-1-i] > remain_b:
                        remain_b = capacityB
                        ans += 1
                break
                    
            if plants[i] > remain_a:
                remain_a = capacityA
                ans += 1
            remain_a -= plants[i]
            if plants[sz-1-i] > remain_b:
                remain_b = capacityB
                ans += 1
            remain_b -= plants[sz-1-i]
        return ans
            
```
