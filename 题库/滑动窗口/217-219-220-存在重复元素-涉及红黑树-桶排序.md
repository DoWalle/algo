#### [217. 存在重复元素](https://leetcode-cn.com/problems/contains-duplicate/)

```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        visited = set()
        for n in nums:
            if n in visited:
                return True
            visited.add(n)
        return False
```

```python
            if window[nums[right]] == 2:
                return True
            while right - left == k:
                window[nums[left]] -= 1
                left += 1
            right += 1
        return False
```
