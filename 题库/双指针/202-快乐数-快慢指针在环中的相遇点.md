题中已经说了，要么变成1，要么无限循环

如何找到循环呢，就用快慢双指针

慢指针每次走一步，快指针每次走两步，有循环的话就能遇到

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        slow = n
        fast = n
        while True:
            slow = self.cnt(slow)
            fast = self.cnt(fast)
            fast = self.cnt(fast)
            print(slow, fast)
            if slow == fast:
                break
        return slow == 1
            
    def cnt(self, num):
        res = 0
        while num > 0:
            res += (num % 10)**2
            num = num // 10
        return res
```
